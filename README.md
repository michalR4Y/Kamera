# Kamera
Progarm obsługuje 3 kamery podłaczone do komputera poprzez ethernet. Zastosowano kamery HIKVISION.
Lewa i prawa kamera toDS-2CD1341G0-I/PL. 
Kamery zostały ustawione na rozdzielczość HD (1920x1080) i jest to na sztywno ustawione w programie.
Centralna kamera DS-2TD2628-3/QA
Posiada wbudowane dwie kamery: normalną w rozdzielczości HD (1920x1080) oraz termowizyjną w rozdzielczości 1280x720P
Każda kamera posiada swój adres IP, który jest na sztywno ustawiony w programie

## Integrate with your tools
Projekt został stworzony pod kontrolą śodowiska Visual Studio 2026.
Aplikacja stworzona z wykorzystanie WPF, C# i NET 

## Description
Celem projektu jest odczyt danych z 3 kamer, które docelowo mają się znaleść na pojeźdie gąsienicowym i wyświetlenie ich na pulpicie komputera.
